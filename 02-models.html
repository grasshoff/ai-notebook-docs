8 Model Aliases

  "aliases": {
    "gemini-pro": "gemini-1.5-pro",

  }
}

# Default Models

The system uses the following default models:

```json
{
  "defaults": {
    "text": "claude-3-5-haiku",
    "audio": "gemini-2.0-pro-exp-02-05"
  }
}
```

# Model Aliases

Model aliases are defined in `settings.json` and provide easy-to-remember names for specific model versions:

```json
{
  "aliases": {
    "gemini-pro": "gemini-1.5-pro",
    "gemini-2": "gemini-2.0-flash-exp",
    "gemini-2-thinking": "gemini-2.0-flash-thinking-exp-01-21",
    "gemini-2-pro": "gemini-2.0-pro-exp-02-05",
    "gemini-audio": "gemini-2.0-pro-exp-02-05",
    "claude-haiku": "claude-3-haiku",
    "claude-sonnet": "claude-3-5-sonnet-latest",
    "whisper": "whisper-large-v3",
    "whisper-turbo": "whisper-large-v3-turbo"
  }
}
```
